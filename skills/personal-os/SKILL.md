# 2026 个人数字化操作系统 (Personal OS)

一个融合全球顶尖管理科学（OKR、GTD、PARA、PDCA）的全链路闭环系统，旨在解决现代人面临的四大核心痛点：**目标涣散、执行无力、资料混乱、难以坚持**。

---

## 第一章：核心设计理念

### 1. 北极星指标 (NSM) & OKR —— 解决"目标涣散"

**核心思想：聚焦 (Focus) 与 层级支撑 (Alignment)**

- **强制单核**：系统强制设定 **唯一** 的年度核心愿景（2026 北极星），这是所有行动的终极指向
- **层级支撑**：北极星之下设立 3-5 个"战略支柱"，所有项目必须服务于这些支柱
- **拒绝伪勤奋**：任务找不到项目、项目找不到支柱 = "无效做功"，应果断砍掉

**结构示例：**
```
🌟 2026 北极星（唯一）
  ├── 🏔 战略支柱1：健康
  ├── 🏔 战略支柱2：财富
  ├── 🏔 战略支柱3：成长
  └── 🏔 战略支柱4：关系
```

### 2. GTD (Getting Things Done) —— 解决"执行无力"

**核心思想：清空大脑 (Mind like Water)**

大脑是用来思考创意的，不是用来记忆琐事的。

- **收集 (Capture)**：用"闪念胶囊"极速捕获一切杂事（灵感、待办、文章），保持"空杯状态"
- **厘清 (Clarify)**：区分"项目"（多步完成）和"任务"（一步完成）
  - 项目 → 【表 02 项目管理】
  - 任务 → 【表 03 行动清单】
- **执行 (Engage)**：使用"情境标签"批量处理任务
  - `@💻 电脑`：必须在书桌前重度办公
  - `@📱 手机`：碎片时间可处理
  - `@🏃 外出`：需要离开家/公司
  - `@🗣 沟通`：需要找人对接、打电话

### 3. PARA 知识管理 —— 解决"资料混乱"

**核心思想：行动导向 (Action-Oriented)**

按"可行动性"而非"话题"组织信息：

| 优先级 | 类别 | 含义 | 示例 |
|--------|------|------|------|
| 1 | Projects | 当前正在做的项目 | 写论文、学Python |
| 2 | Areas | 需要持续维护的领域 | 健康、财务、成长 |
| 3 | Resources | 未来可能用到的资料 | 模板、教程、文章 |
| 4 | Archives | 已完成或不再活跃 | 过去的项目存档 |

### 4. 量化自我 (Quantified Self) —— 解决"难以坚持"

**核心思想：数据驱动成长**

- 每日记录：早起、运动、阅读、状态评分(1-5)
- 能量分计算（自动）：基础分50 + 早起+10 + 阅读+10 + 运动达标+20 + 心情好+10 = 最高100分
- 热力图可视化：类似 GitHub 的自律热力图

### 5. PDCA 循环 —— 解决"盲目行动"

```
Plan (计划) → Do (执行) → Check (检查) → Act (调整)
     ↑                                           ↓
     └───────────────────────────────────────────┘
```

- **Plan**：设定北极星和战略支柱
- **Do**：执行项目和任务
- **Check**：查看仪表盘、检查进度
- **Act**：周复盘、调整方向

## 第二章：系统架构（六层结构）

### 战略层：【01 🎯 愿景与OKR】

| 字段 | 说明 |
|------|------|
| 层级类型 | `🌟 2026 北极星`（全年唯一）或 `🏔 战略支柱`（3-5个） |
| 2026 进度 | 自动计算关联项目进度平均值 |

### 战术层：【02 🚀 项目管理】

| 字段 | 说明 |
|------|------|
| 关联 OKR | 项目服务于哪个战略支柱和北极星 |
| ⏳ 时间进度 | 自动计算（开始日期→截止日期） |
| 🚦 偏差判定 | 🟢 正常（实际>时间流逝）/ 🔴 滞后（实际<时间流逝） |
| 🏷 项目名片 | 自动生成"名称+进度+红绿灯" |

### 执行层：【03 ✅ 行动清单】

| 字段 | 说明 |
|------|------|
| 完成 (状态) | 点击"是"即完成，自动记录时间并驱动进度 |
| 情境标签 | `@💻电脑` `@📱手机` `@🏃外出` `@🗣沟通` |
| 计划执行日 | 任务安排在哪天执行 |
| 关联项目 | 任务属于哪个项目 |

### 输入层：【04 📥 万能收集箱】

- **⚡️ 闪念胶囊**：极速录入入口
- 建议配置：生成链接/二维码，添加到手机桌面快捷方式
- 用途：捕获一切一闪而过的念头，不打断当前工作心流

### 能量层：【06 ❤️ 习惯与记录】

| 字段 | 说明 |
|------|------|
| 📊 状态评分 | 每晚睡前填入，1-5分，仪表盘热力图数据源 |
| ⚡️ 能量分 | 自动计算：50基础分 + 打卡加分 |

## 第三章：系统自动化逻辑

### 进度的"蝴蝶效应"

```
任务完成 → 项目进度↑ → 战略支柱达成率↑ → 北极星进度↑
```

**你当下的每一个微小行动，都在直观地推动人生目标。**

### 智能防错机制

- **空项目保护**：新项目未拆解任务时，进度锁定为 0%
- **逾期预警**：每日凌晨扫描逾期任务，红色高亮标记

### 机器人助理

- **周复盘提醒**：每周日晚 20:00 自动发送
- 自动在【复盘日志】创建空白记录

## 第五章：SOP 标准作业程序

### 初始化：第一小时

1. **清空演示数据**：删除示例数据（勿删公式列）
2. **确立北极星**：问自己"如果2026年只做成一件事，那是什么？"
3. **部署闪念胶囊**：获取分享链接，保存到手机最顺手位置
4. **启动本周**：建立1-3个当前项目，列出明天3-5件小事

### 日工作流：每天只需 10 分钟

**遵循"早晨看，白天做，晚上清"原则**

#### ☀️ 早晨（启动期 - 2分钟）
1. 打开"🌟 2026 驾驶舱"仪表盘
2. 扫一眼左上角"2026 达成率"
3. 盯着"⚡️ 今日必做"列表 → 这是今天的战场

#### 🌤 白天（执行期 - 专注流）
1. 执行任务，利用"情境标签"批量处理
2. 遇到打断（灵感/指令/想法）→ 打开"闪念胶囊"5秒录入 → 立即回到手头工作

#### 🌙 晚上（关闭期 - 8分钟）
1. **清空收集箱**：处理今天收集的杂事（转为任务/存资料/删除）
2. **打卡复盘**：
   - 在【今日必做】把完成任务点成"是"
   - 在【习惯与记录】给今天状态打分(1-5)
3. **关机**：看着热力图变深，安心睡觉

### 周工作流：每周日"CEO 会议"（20分钟）

1. **响应提醒**：收到机器人复盘提醒
2. **检查数据**：
   - 看本周"生命律动"曲线：状态下滑了吗？
   - 看【02 项目表】：有哪些🔴滞后红灯？
3. **填写日志**：进入【07 复盘日志】，记录评分、高光时刻、反思

## 常见问题

### Q: 任务找不到项目怎么办？
**A:** 这个任务可能是"伪勤奋"。问自己：它服务于我的北极星吗？如果不是，果断砍掉。

### Q: 项目进度不更新？
**A:** 检查是否正确关联了OKR和北极星，确保任务有"关联项目"字段。

### Q: 热力图颜色很浅？
**A:** 说明能量分低。检查是否坚持了早起、运动、阅读习惯。

### Q: 一天任务太多做不完？
**A:** 遵循"每天3-5件小事"原则。把多余任务移到其他日期或降级为"也许某天"。

## 帮助用户使用此系统

当用户需要帮助时，按以下方式引导：

1. **初始化引导**：帮助用户确立年度北极星和战略支柱
2. **项目拆解**：将大目标拆解为可执行的项目和任务
3. **情境标签建议**：根据任务性质推荐合适的情境标签
4. **复盘提醒**：协助用户进行日/周复盘
5. **数据解读**：帮助用户理解仪表盘数据的含义

**核心原则：让系统服务于人，而不是人被系统奴役。**
