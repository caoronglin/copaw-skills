# 高德地图 MCP 技能

## 功能说明

高德地图 MCP Server 提供全面的地图服务：

- 📍 **IP定位**：根据IP地址定位
- 🔍 **POI搜索**：搜索周边兴趣点
- 🗺️ **地理编码**：地址与坐标相互转换
- 🚗 **路线规划**：驾车、步行、公交路线
- 🌤️ **天气查询**：指定地点的天气信息
- 🔄 **坐标转换**：不同坐标系之间的转换

## 工具列表

### 1. locate_ip - IP定位

**功能**：根据IP地址获取地理位置

**参数说明**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| ip | string | ❌ | IP地址，为空则使用请求方IP |

**返回信息**：
- 省份、城市
- 经纬度坐标
- ISP信息

**使用示例**：
```json
{
  "ip": "114.247.50.2"
}
```

### 2. search_poi - POI搜索

**功能**：搜索指定类型的兴趣点

**参数**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| keywords | string | ✅ | 搜索关键词 |
| location | string | ❌ | 中心点坐标 |
| radius | integer | ❌ | 搜索半径（米） |
| types | string | ❌ | POI类型 |
| offset | integer | ❌ | 每页记录数 |

**POI类型代码**：
- 餐饮服务：050000
- 酒店住宿：060000
- 购物服务：060100
- 医疗保健：090000
- 交通设施：150000

### 3. geocode - 地理编码

**功能**：将地址转换为经纬度坐标

**参数**：
- `address`: 结构化地址

**返回**：
- 经纬度坐标
- 精确度等级
- 匹配的地址

### 4. regeocode - 逆地理编码

**功能**：将坐标转换为地址信息

**参数**：
- `location`: 经纬度坐标
- `radius`: 搜索半径（米）

**返回**：
- 省市区信息
- 街道门牌号
- 周边POI

### 5. get_driving_route - 驾车路线规划

**功能**：规划驾车路线

**参数**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| origin | string | ✅ | 起点坐标 |
| destination | string | ✅ | 终点坐标 |
| strategy | integer | ❌ | 路径策略 |

**策略代码**：
- 0：速度优先
- 1：费用优先
- 2：距离优先
- 4：躲避拥堵

**返回信息**：
- 总距离、总时间
- 分段路线详情
- 路况信息

### 6. get_walking_route - 步行路线规划

**功能**：规划步行路线

**参数**：
- `origin`: 起点坐标
- `destination`: 终点坐标

### 7. get_transit_route - 公交路线规划

**功能**：规划公共交通路线

**参数**：
- `origin`: 起点
- `destination`: 终点
- `city`: 城市名称

## 配置方式

### 获取API Key
1. 访问高德开放平台：https://lbs.amap.com
2. 注册并创建应用
3. 获取Web服务API Key

### MCP 配置
```json
{
  "mcpServers": {
    "amap": {
      "command": "uvx",
      "args": ["amap-mcp-server"],
      "env": {
        "AMAP_MAPS_API_KEY": "your_api_key_here"
      }
    }
  }
}
```

### SSE 传输模式
```bash
export AMAP_MAPS_API_KEY=your_api_key
uvx amap-mcp-server --transport sse --port 8000
```

## 坐标格式说明

### 支持的坐标系
- GCJ-02（国测局坐标，高德使用）
- WGS-84（GPS原始坐标）
- BD-09（百度坐标）

### 坐标格式
```
经度,纬度
例如：116.397428,39.90923
```

## 最佳实践

### 1. POI搜索优化
```python
# 指定搜索范围
search_poi(
    keywords="餐厅",
    location="116.397,39.909",  # 中心点
    radius=1000  # 1公里范围
)

# 按类型筛选
search_poi(
    types="050000",  # 餐饮服务
    location="116.397,39.909"
)
```

### 2. 路线规划策略
```
日常通勤：躲避拥堵（strategy=4）
长途出行：速度优先（strategy=0）
省钱路线：费用优先（strategy=1）
```

### 3. 坐标精度
- 一般查询：保留6位小数
- 精确定位：保留8位小数

## 常见问题

### Q: 为什么坐标有偏差？
**A**: 高德使用GCJ-02坐标系，与GPS原始坐标(WGS-84)有偏差

### Q: POI搜索结果太少？
**A**: 尝试：
- 放宽搜索范围
- 使用更通用的关键词
- 指定正确的类型代码

### Q: 如何提高配额？
**A**: 完成企业认证或购买增值服务
