# 发现报告 MCP 技能

## 功能说明

发现报告(www.fxbaogao.com)是专业的研报搜索平台：

- 📊 **行业分析报告**：各行业发展趋势、市场规模、竞争格局分析
- 🏢 **公司研究报告**：上市公司深度研究、财务分析、估值报告
- 📈 **宏观策略报告**：宏观经济分析、政策解读、投资策略
- 💰 **财报分析**：季度/年度财报解读、财务数据分析
- 📋 **招股说明书**：IPO招股书、上市材料
- 📑 **专项研究**：尽职调查、定制研究报告

## 工具列表

### 1. search_reports - 研报搜索

**功能**：搜索专业研究报告

**参数说明**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| query | string | ✅ | 搜索关键词 |
| report_type | string | ❌ | 报告类型：行业/公司/宏观/财报 |
| industry | string | ❌ | 行业分类 |
| date_range | string | ❌ | 时间范围 |
| limit | integer | ❌ | 返回数量，默认10 |

**使用示例**：
```json
{
  "query": "人工智能",
  "report_type": "行业",
  "date_range": "近一年",
  "limit": 10
}
```

**返回信息**：
- 报告标题
- 作者/机构
- 发布日期
- 报告摘要
- PDF下载链接

### 2. get_report_detail - 报告详情

**功能**：获取报告的详细信息

**参数**：
- `report_id`: 报告ID

**返回**：
- 完整报告内容
- 核心观点
- 关键数据
- 图表信息

### 3. get_industry_reports - 行业报告

**功能**：获取特定行业的所有报告

**参数**：
- `industry`: 行业名称
- `sub_industry`: 子行业（可选）

### 4. get_company_reports - 公司报告

**功能**：获取特定公司的研究报告

**参数**：
- `company_name`: 公司名称
- `stock_code`: 股票代码（可选）

## 最佳实践

### 1. 精准搜索
```python
# 使用具体的行业名称
search_reports(query="新能源汽车动力电池", report_type="行业")

# 使用公司全称或股票代码
get_company_reports(company_name="贵州茅台", stock_code="600519")
```

### 2. 组合使用
```python
# 先搜索行业报告
industry_reports = search_reports("人工智能")

# 再搜索重点公司报告
company_reports = get_company_reports("商汤科技")

# 综合分析
```

### 3. 时间筛选
```python
# 获取最新报告
search_reports(query="新能源", date_range="近一个月")

# 获取历史报告进行对比
search_reports(query="新能源", date_range="2023年")
```
