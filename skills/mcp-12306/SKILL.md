# 12306 火车票查询 MCP 技能

## 功能说明

12306 MCP Server 提供中国铁路12306官方网站的火车票查询功能：

- 🚄 **车次查询**：查询两站之间的所有车次
- 💺 **余票查询**：实时查询各座位类型的余票情况
- 💰 **票价查询**：获取各座位类型的票价信息
- ⏰ **时刻表**：查询列车发车和到达时间
- 🔄 **中转查询**：支持中转方案推荐

## 工具列表

### 1. search_trains - 车次查询

**功能描述**：查询指定日期两站之间的所有车次

**参数说明**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| from_station | string | ✅ | 出发站（站名或电报码） |
| to_station | string | ✅ | 到达站（站名或电报码） |
| date | string | ✅ | 出发日期（YYYY-MM-DD） |
| purpose_codes | string | ❌ | 票种：ADULT（成人）/ STUDENT（学生） |

**使用示例**：
```json
{
  "from_station": "北京",
  "to_station": "上海",
  "date": "2026-03-10",
  "purpose_codes": "ADULT"
}
```

### 2. query_ticket - 余票查询

**功能描述**：查询指定车次的余票情况

**参数说明**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| train_no | string | ✅ | 车次号 |
| from_station | string | ✅ | 出发站 |
| to_station | string | ✅ | 到达站 |
| date | string | ✅ | 出发日期 |

**返回字段**：
- 各座位类型的余票数量（商务座、一等座、二等座等）
- 座位类型代码和名称
- 是否可购买标志

### 3. get_station_code - 站点代码查询

**功能描述**：获取站点名称对应的电报码

**参数**：
- `station_name`: 站点名称

**用途**：车次查询需要使用站点电报码

## 常见问题

### Q: 查询返回空结果？
**A**: 可能原因：
- 日期超出可查询范围（15天内）
- 站点名称不正确
- 该日期无车次运行

### Q: 如何区分站点代码？
**A**: 使用 get_station_code 工具查询：
- 北京站 → BJP
- 北京西站 → BXP
- 上海站 → SHH
- 上海虹桥站 → AOHHQ

### Q: 余票显示"有"但无法购买？
**A**: 可能原因：
- 网络延迟导致数据未同步
- 车票已被其他用户锁定
- 需要登录12306账号才能购买
