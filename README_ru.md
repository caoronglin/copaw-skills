## 🎯 8 Основных Возможностей

### 1️⃣ Информация и Поиск ⭐⭐⭐⭐⭐
- **Tavily** - Поиск в интернете в реальном времени
- **Metaso** - ИИ-поиск (оптимизирован для науки)
- **DeepWiki** - Запросы к Википедии
- **Fetch** - Извлечение веб-контента

### 2️⃣ Академические Исследования ⭐⭐⭐⭐⭐
- **arXiv** - Поиск и загрузка статей
- **BioMCP** - Биоинформатика (гены, белки, испытания)
- **FXBaogao** - Отраслевые исследования
- **Context7** - Техническая документация

### 3️⃣ Обработка Данных ⭐⭐⭐⭐⭐
- **Chart** - 25+ типов графиков
- **XLSX** - Обработка Excel
- **DOCX** - Работа с Word
- **PDF** - Обработка PDF и OCR

### 4️⃣ Мультимедиа и Творчество ⭐⭐⭐⭐
- **Image Recognition** - OCR (109 языков)
- **ModelScope** - ИИ: текст в изображение
- **PPTX** - Создание презентаций

### 5️⃣ Бытовые Сервисы ⭐⭐⭐⭐⭐
- **12306** - Поиск ж/д билетов
- **Amap** - Поиск мест, маршруты
- **Cron** - Запланированные задачи

### 6️⃣ Инструменты Разработчика ⭐⭐⭐⭐
- **GitHub** - Управление репозиториями
- **Scrapling** - Профессиональный парсинг
- **Markdownify** - Конвертация в Markdown

### 7️⃣ Интеграция с Feishu ⭐⭐⭐⭐⭐
- **Feishu** - Сообщения, документы, календарь

### 8️⃣ Управление Знаниями ⭐⭐⭐⭐⭐
- **Personal-OS** - Персональная ОС (OKR+GTD+PARA)
- **News** - Агрегация новостей
- **Deep Research** - Глубокие исследования

## 🚀 Быстрый Старт

### Способ 1: Автоматическое Создание

Скажите вашему ИИ-ассистенту:

```
Помоги создать конфигурацию навыков CoPaw:
- Документы: docx, xlsx, pptx, pdf
- Наука: arxiv, biomcp, fxbaogao
- Визуализация: chart
- Быт: 12306, amap
- Разработка: github, scrapling

Используй stdio MCP, секреты из переменных окружения.
```

### Способ 2: Ручная Настройка

#### 1. Установка Зависимостей

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
cd copaw-skills
uv pip install -r requirements.txt
```

#### 2. Настройка Окружения

Создайте `.env`:

```bash
# Погода
QWEATHER_API_KEY=ваш_ключ

# Поиск
TAVILY_API_KEY=ваш_ключ

# Карты
AMAP_API_KEY=ваш_ключ

# GitHub
GITHUB_TOKEN=ghp_ваш_токен

# ModelScope
MODELSCOPE_API_TOKEN=ваш_токен
```

## 💡 Примеры Использования

### Наука
```python
arxiv.search(query="CRISPR")
biomcp.search_gene("TP53")
fxbaogao.search("рынок электромобилей")
```

### Данные
```python
chart.create_line_chart(data)
xlsx.read_excel("data.xlsx")
docx.create_document("отчет.docx")
```

### Быт
```python
train.search(from="Пекин", to="Янлин")
amap.get_weather(location="Янлин")
cron.create_task("Ежедневная пробежка в 8:00")
```
