## 🚀 使用方式

### 1. 克隆仓库
```bash
git clone https://github.com/caoronglin/copaw-skills.git
cd copaw-skills
```

### 2. 安装依赖
```bash
uv pip install -r requirements.txt
```

### 3. 配置环境变量
```bash
cp .env.example .env
# 编辑 .env 填写真实密钥
```

### 4. 让 AI 配置
对你的 AI 助手说：
```
帮我配置 CoPaw skills，参考这个仓库：
https://github.com/caoronglin/copaw-skills

需要的技能：
- 文档：docx, xlsx, pptx, pdf
- 学术：arxiv, biomcp, fxbaogao
- 图表：mcp-chart
- 生活：mcp-12306, mcp-amap
- 开发：mcp-github, scrapling
- 知识：deep-research, personal-os

要求：
1. 使用 stdio MCP 配置
2. 密钥从环境变量读取
3. 创建 .env.example
```
